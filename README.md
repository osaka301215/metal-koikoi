# 金属イオンこいこい

金属イオンの価数を使った花札風ゲーム（こいこい）。

## 遊び方

`index.html` をブラウザで開くか、デプロイされたURLにアクセスしてください。

## 役一覧

- 塩化物沈殿 (Ag + Pb): 2点
- アルカリ金属 (Li + Na + K): 5点
- アルカリ土類金属 (Mg + Ca + Ba): 5点
- 硫酸塩沈殿 (Ca + Ba + Pb): 5点
- 硫化物沈殿(酸性) (Cu + Pb + Ag): 5点
- トタンとブリキ (Fe + Zn + Sn): 5点
- ステンレス (Fe + Cr + Ni): 5点
- ジュラルミン (Al + Cu + Mg): 5点
- 不動態 (Fe + Al + Cr): 5点
- アンモニア錯体 (Cu + Ag + Zn): 5点
- 炎色反応 (3つで5点 / 6つで10点)
- 銅合金 (Cu + Zn + Sn + Ni): 7点
- 両性元素 (Al + Zn + Sn + Pb): 7点
- 貴金属 (Au + Pt + Ag): 5点

## ルール

- 上がり最低5点
- こいこい1回ごとに上がり時+3点ボーナス
